## CHROME WEB STORE

**Nome da extensão (máx 45 caracteres):**
SEI+Trello (Fork Atualizado)

**Descrição curta (máx 132 caracteres):**
Integra o SEI (Sistema Eletrônico de Informações) com o Trello. Fork atualizado para SEI 4.x e Chrome Manifest V3.

**Descrição completa:**
Permite visualizar e criar cartões do Trello diretamente na interface do SEI (Sistema Eletrônico de Informações), sem precisar alternar entre os sistemas.

✅ Compatível com SEI 4.x
✅ Compatível com Chrome Manifest V3
✅ Cores dos cartões se adaptam automaticamente ao tema do SEI

FUNCIONALIDADES:
• Visualiza cartões do Trello na lista de processos do SEI
• Cria cartões Trello a partir de qualquer processo
• Edita nome, descrição, etiquetas, checklists e data de vencimento
• Filtra processos por quadro, lista ou etiqueta
• Cores dos cartões acompanham o tema de cor escolhido pelo usuário no SEI

CONFIGURAÇÃO:
Após instalar, acesse as Opções da extensão e informe:
1. Sua Trello APP-KEY (obtenha em trello.com/app-key)
2. Seu Trello TOKEN
3. Nome do quadro padrão
4. Nome da lista padrão

Este é um fork mantido pela comunidade do projeto original de Luís Carlos Reischak Júnior (github.com/luiscrjunior/sei-trello), atualizado para funcionar com as versões mais recentes do SEI e do Chrome.

**Categoria:** Ferramentas de produtividade

**Idioma principal:** Português (Brasil)

**Site:** https://github.com/awlmdnitce/sei-trello
